# UI_GUIDELINES — معايير الواجهة

## 1. القاعدة الأولى — الواجهة لا تكذب

**واجهة لا تُظهر ميزة غير موجودة.** واجهة تعرض «استوديو الصياغة» بلا محرك صياغة، ومربع بحث لا يبحث — تُنشئ ثقة كاذبة. المحامي الذي يثق بواجهة كاذبة يخسر قضية.

هذا **الخرق الأخطر في الواجهة الحالية**:

| العنصر | ما يوحي به | ما هو فعلًا |
|---|---|---|
| «ابحث في المعرفة» | بحث في قاعدة المعرفة | `Array.filter()` على 1000 صف في المتصفح |
| «استوديو الصياغة» | توليد مستندات | مُنتقي قضايا بلا توليد |
| «الرسم المعرفي» | رسم علاقات | جدول `relationships` = **0 صف** |

**القاعدة:** الميزة غير المُنفَّذة تُعرَض **معطّلة ومعلَّلة** («قيد التطوير — لا يوجد محرك استرجاع بعد») أو **لا تُعرَض**. لا تُعرَض عاملةً وهي ليست كذلك.

## 2. RTL أصلي

العربية هي اللغة الأولى، لا ترجمة. الاتجاه `rtl` أصلي (`pr-10 pl-4` للأيقونة يمينًا). التواريخ بـ `toLocaleString('ar-KW')`.

## 3. حالة الفراغ تقول الحقيقة

قاعدة المعرفة فارغة اليوم. الشاشة يجب أن تقول **«لا توجد معرفة قانونية بعد — ارفع أول مصدر»**، لا أن تعرض جدولًا فارغًا يوحي بعطل، ولا أن تعرض الكائنين التجريبيين كأنهما محتوى.

## 4. حالة السند ظاهرة دائمًا

كل كائن معرفي يُعرَض **مقرونًا بحالة توثيقه**:

| الحالة | العرض |
|---|---|
| `source_verified` | أخضر |
| `operationally_accepted` | كهرماني — **معتمد دون مراجعة بندية** |
| `machine_pending_human` | كهرماني — ينتظر مراجعة |
| `historical_only` | **أحمر — لا يُستند إليه كقانون نافذ** |

`historical_only` **لا يُعرَض كنتيجة عادية أبدًا.** المحامي الذي يستشهد بمبدأ تاريخي ظنًا أنه نافذ يخسر قضية.

## 5. بوابة السند مرئية

حالة التغطية ظاهرة في ملف القضية:
- 🟢 `ready_for_grounded_draft`
- 🟡 `blocked_missing_authorities` + **قائمة الناقص صراحةً**

المستخدم يرى **لماذا** الصياغة مقفلة، لا مجرد أنها مقفلة. هذا مُنفَّذ فعلًا وهو سلوك جيد — حافظ عليه.

## 6. لغة التصميم القائمة

| العنصر | القيمة |
|---|---|
| الخلفية | `slate-50` / بطاقات بيضاء |
| الرأسية الداكنة | `slate-950` |
| التمييز | `amber-400` |
| النجاح | `emerald-500` |
| الاستدارة | `rounded-2xl` / `rounded-3xl` |
| الأيقونات | `lucide-react` |
| الشبكة | `grid-cols-[1fr_360px]` (محتوى + شريط جانبي) |

## 7. لا حالة سرية

`refresh()` يجلب كل شيء دفعة واحدة (`Promise.all`). مقبول اليوم (البيانات صفرية).
**لن يصمد** عند آلاف الكائنات — `/api/documents?limit=1000` سيجلب ميغابايتات إلى المتصفح. **يجب** التحول إلى ترقيم صفحات + بحث من الخادم عند بناء [[RETRIEVAL_ENGINE]].

## 8. static export — قيد معماري

`next.config.mjs` فيه `output: 'export'`. لا SSR، لا API routes، لا server actions في `web/`.
**كل منطق الخادم يعيش في `admin/` (FastAPI).** لا تُضف منطقًا في `web/` يتطلب خادم Next — سيُبنى ولن يعمل.
