# DRAFTING_ENGINE — محرك الصياغة

> **الحالة: ❌ غير موجود.** جدول `case_drafts` قائم ولا يكتب فيه أي كود.
> `DraftingView` في الواجهة **مُنتقي قضايا** فقط.
> هذا هو **المنتج النهائي** للمشروع، وهو آخر ما يُبنى — لا أوله.

## 1. الشرط المسبق المطلق

```text
لا يبدأ بناء محرك الصياغة قبل:
  ✅ وجود معرفة قانونية حقيقية في قاعدة البيانات   ← اليوم: 0
  ✅ وجود محرك استرجاع يعمل                        ← اليوم: غير موجود
  ✅ وجود تضمين دلالي حقيقي                        ← اليوم: hashing trick
```

**بناء الصياغة اليوم يعني توليد مذكرة قانونية بأسانيد غير موجودة.** هذا الخطر الوحيد غير المقبول في المشروع. مذكرة خاطئة تُقدَّم لمحكمة تُسقط حقًا.

## 2. المخرجات المستهدفة

رأي قانوني · صحيفة دعوى · مذكرة دفاع · صحيفة استئناف أو تمييز · طلب أو تظلم · إنذار · قائمة إجراءات ومستندات.

## 3. شروط بدء الصياغة

قبل أي توليد، يجب أن يكون محددًا: صفة كل طرف · الوقائع الثابتة **وغير الثابتة** · الطلب النهائي · المحكمة والمرحلة · الفرع/الموضوع/المسألة · النصوص النافذة · المبادئ المطابقة · المستندات المتاحة **والناقصة**.

## 4. سلسلة السند — القاعدة الحاكمة

**كل فقرة قانونية** في المخرج تجب أن ترتبط بأحد:

```text
نص تشريعي رسمي
أو  حكم / مبدأ قضائي موثق
أو  قاعدة جامعة مرتبطة بمصادرها الداعمة
```

**فقرة بلا سند = المستند يُرفض بأكمله.** لا تُحذف الفقرة بصمت، ولا تُخرَّج بتحفظ. يُرفض المستند ويُعلَن سبب الرفض.

## 5. ما يُمنع منعًا باتًا

| ممنوع | لماذا |
|---|---|
| اختلاق رقم مادة أو طعن أو جلسة | تضليل محكمة |
| الاستناد إلى `historical_only` كقانون نافذ | المبدأ 4 مقيد بـ«قبل القانون المدني الجديد» |
| خلط القواعد السنية والجعفرية | بطلان مذهبي |
| توسيع مبدأ مختصر إلى تسبيب كامل | اختلاق تسبيب لم تقله المحكمة |
| تجاوز بوابة التغطية | نقض الغرض من المشروع |
| إخراج مستند «تقريبي» بتحذير | التحذير لا يُقرأ. المستند يُقدَّم للمحكمة. |

## 6. المخرج الإلزامي

كل مسودة تحمل معها **تقرير أسانيد** (`case_drafts.authority_report` — العمود موجود):

```json
{
  "paragraphs": [
    {"para": 1, "authorities": ["LEG-KW-51-1984-ART-338"], "status": "grounded"},
    {"para": 2, "authorities": ["JUR-PS-JURISDICTION-00013"], "status": "grounded"}
  ],
  "ungrounded_paragraphs": [],
  "missing_documents": ["شهادة الميلاد"],
  "temporal_warnings": ["JUR-PS-JURISDICTION-00004 مقيد تاريخيًا"],
  "drafting_status": "ready_for_grounded_draft"
}
```

**`ungrounded_paragraphs` غير فارغة ⇒ المستند لا يُسلَّم.**

## 7. الحالات

| الحالة | المعنى |
|---|---|
| `blocked_missing_authorities` | التغطية ناقصة — الافتراضي في المخطط |
| `ready_for_grounded_draft` | البوابة مفتوحة |
| `draft_generated` | مسودة مُنتَجة بسند كامل |
| `human_reviewed` | راجعها محامٍ — **الحالة الوحيدة الصالحة للتقديم لمحكمة** |

## 8. الإصدارات

`case_drafts` عليه قيد `UNIQUE(case_id, draft_type, version)`. المسودات **تُصدَّر ولا تُدهس**. لا يُحذف تاريخ الصياغة.

## 9. الدور النهائي

النظام **لا يُنتج مستندًا نهائيًا للمحكمة**. ينتج **مسودة مُسنَدة** يراجعها محامٍ ويتحمل مسؤوليتها. هذا حد أخلاقي ومهني، لا حد تقني.
