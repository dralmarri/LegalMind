# صوت العدالة (LegalMind) — المرجع القانوني الكويتي

منصة قانونية متكاملة تعمل بالكامل في المتصفح: مكتبة مصادر قانونية كويتية + محرك بحث عربي ذكي + مستشار قانوني بالذكاء الاصطناعي يجيب **حصرياً من مصادرك المرفوعة** مع توثيق كل معلومة بمصدرها.

## المكونات

### 1. مكتبة المصادر (ثلاثة أنواع)
- **أحكام ومبادئ قضائية** — أحكام محكمة التمييز وغيرها من المحاكم الكويتية، مع بيانات الطعن والدائرة والجلسة.
- **نصوص قانونية** — القوانين والمراسيم واللوائح؛ يُفصَّل النص تلقائياً إلى مواد (كل سطر يبدأ بـ "مادة 1" أو "المادة (1)" يبدأ مادة جديدة)، وتصبح كل مادة وحدة بحث واستشهاد مستقلة.
- **نماذج قانونية** — الصيغ المتبعة أمام المحاكم الكويتية: صحف دعاوى، لوائح استئناف، صحف طعن بالتمييز، مذكرات دفاع، إنذارات عدلية، أوامر أداء، عقود، توكيلات...

### 2. التصنيف والربط الآلي
- **استخلاص المواد**: عند حفظ حكم، تُستخلص تلقائياً كل الإشارات إلى المواد القانونية (المادة 227 من القانون المدني...) وتُربط بالنصوص القانونية المرفوعة في المكتبة.
- **الربط العكسي**: عند تصفح أي مادة من قانون مرفوع، تظهر الأحكام التي طبّقتها.
- **المواضيع القانونية**: تصنيف تلقائي (مع إمكانية التعديل اليدوي) على ~30 موضوعاً قانونياً (العقود، الإثبات، الفصل التعسفي، الشيك، الحضانة...) وربط كل المصادر بها.
- **سجل القوانين الكويتية**: التعرف الذكي على القوانين الأساسية (المدني 67/1980، المرافعات 38/1980، التجارة 68/1980، الجزاء 16/1960، الأحوال الشخصية 51/1984، العمل 6/2010...).

### 3. المستشار القانوني (صفر هلوسة)
- **استرجاع دلالي**: تطبيع عربي كامل (همزات، تاء مربوطة، تشكيل، أرقام هندية) + تجذيع خفيف + ترتيب BM25 على مستوى المقاطع والمواد.
- **إجابة مقيّدة بالمصادر**: يتقمص النموذج شخصية محامٍ كويتي خبير، ويُلزَم بالإجابة حصرياً من المقاطع المسترجعة، وتوثيق كل معلومة بـ 【N】، والتصريح بالنقص إن لم تكفِ المصادر.
- **فحص مضاد للهلوسة**: بعد كل إجابة يجري تحقق آلي من أن كل اقتباس حرفي «...» موجود فعلاً في المصادر (يُميَّز الاقتباس المتحقق بالأخضر وغير المتحقق بتحذير)، مع بيان المصادر المستشهد بها.
- **وضع بدون مفتاح API**: استرجاع استنادي خالص — يعرض المقاطع الأصلية حرفياً (دقة 100%).

## التشغيل

```bash
npm start          # يشغل خادم الملفات الثابتة على المنفذ 5000
```

أو انشر الملفات على أي استضافة ثابتة (الموقع لا يحتاج خادماً خاصاً — كل شيء يعمل في المتصفح).

## الإعداد

1. افتح تبويب **الإعدادات** وأدخل مفتاح Claude API من [console.anthropic.com](https://console.anthropic.com) (يُحفظ محلياً على جهازك فقط).
2. ارفع مصادرك من تبويب **إضافة مصدر**.
3. اسأل من تبويب **المستشار القانوني**.

## الخصوصية والبيانات

- كل البيانات (المصادر + المفتاح) محفوظة محلياً في متصفحك (IndexedDB + localStorage) ولا تغادر جهازك إلا عند سؤال المستشار الذكي، حيث تُرسل المقاطع ذات الصلة إلى Anthropic API مباشرة.
- صدّر نسخة احتياطية دورياً من تبويب الإعدادات (يدعم الاستيراد صيغ التصدير القديمة والجديدة).
- بيانات الإصدار السابق تُرحَّل تلقائياً عند أول فتح.

## بنية الملفات

```
index.html      الواجهة (تبويبات: مستشار / مكتبة / إضافة / إعدادات)
style.css       التنسيق (فاتح/داكن)
js/arabic.js    تطبيع وتقطيع النص العربي
js/extract.js   استخلاص المواد + تفصيل القوانين + تصنيف المواضيع + سجل القوانين الكويتية
js/db.js        التخزين (IndexedDB) مع الترحيل التلقائي
js/search.js    محرك الاسترجاع (BM25 على المقاطع)
js/ai.js        المستشار الذكي (Claude API) + التحقق من الاقتباسات
js/app.js       منطق الواجهة
server.js       خادم ملفات ثابتة اختياري للتطوير المحلي
```
