# LegalMind Reasoning Trace v1.0

## 1. الغرض

تسجيل المسار القانوني القابل للمراجعة الذي أدى إلى النتيجة، دون الاعتماد على سرد حر غير موثق.

## 2. البنية


## 3. ما يعرض للمستخدم

- المسألة القانونية المحددة.
- المواد والمبادئ المستخدمة.
- القيود الزمنية.
- أهم المصادر المستبعدة وسبب الاستبعاد.
- مواطن النقص أو التعارض.

## 4. ما لا يعد سندًا

وسوم الاسترجاع أو درجة التشابه أو نص النموذج ليست سندًا قانونيًا. وظيفتها تفسير آلية الاختيار فقط.

## 5. مثال استرجاع

سؤال: هل تختص دائرة الأحوال الشخصية بطلب الحاضنة تسليم جوازات سفر أولادها؟

```text
الفرع: أحوال شخصية
العنوان: اختصاص دائرة الأحوال الشخصية
المسألة الدقيقة: جوازات سفر الأولاد
المصدر الحاسم: JUR-PS-JURISDICTION-00022
النتيجة: الطلب لا يتعلق بأحكام قانون الأحوال الشخصية وفق المبدأ، ويعامل كمسألة مدنية.
المستبعد: المبادئ العامة عن شمول الاختصاص؛ لأنها مقيدة بطبيعة الطلب وبالحاجة إلى تطبيق قانون الأحوال الشخصية.
```
