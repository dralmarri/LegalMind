# مواصفة إخراج نتائج الفرز

كل تقرير مجموعة يتضمن: عدد المبادئ، نطاق الترقيم، حالة الاكتمال، معرف الدفعة، التصنيف الموضوعي، الهيكل، المسائل الدقيقة، القواعد الجامعة، الإحالات المتقاطعة، ملاحظات التوثيق، ملاحظات التحرير، والعناصر التي تحتاج مراجعة.

كل قاعدة جامعة تعرض مصادرها الداعمة بروابط داخلية قابلة للفتح.

كل مسألة توثيق تعرض في جدول قابل للتعديل بالحقول: معرف المسألة، الكائن، الحالة، الوصف، الدليل، الإجراء المقترح، الحل، وآخر تحديث.

كل مبدأ يحمل anchor ثابتًا ومعرفًا فريدًا وبيانات الطعن والدائرة والجلسة والمصدر والمواد والقيد الزمني ونطاق الاستخدام وعدم الاستخدام.
