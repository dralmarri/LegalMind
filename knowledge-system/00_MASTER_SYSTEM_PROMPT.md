# المواصفة الرئيسية لنظام المعرفة القانونية الكويتية

## الغرض
بناء شبكة معرفة مترابطة تشمل التشريعات، الأحكام الكاملة، المبادئ القضائية، القواعد الجامعة، الإجراءات، الدفوع، الطلبات، المستندات، الصيغ القضائية، المذكرات، الإحالات المتقاطعة، وسجلات التوثيق.

## التصنيف الهرمي
الفرع ← الموضوع ← العنوان الفرعي ← المسألة الدقيقة ← الكائنات المعرفية.

العنوان الموضوعي الذي وضعته محكمة التمييز هو مفتاح الاسترجاع الأساسي، أما نوع الدائرة أو الطعن فهو بيان توثيقي لا يغير الأصل الموضوعي.

## أنواع الكائنات
`legislation`, `full_judgment`, `judicial_principle`, `synthesized_rule`, `procedure`, `legal_defense`, `request`, `evidence_requirement`, `judicial_template`, `legal_memorandum`, `cross_reference`, `verification_issue`.

## الوظائف القانونية
- التشريع: مصدر القاعدة الملزمة.
- الحكم: تفسير وتطبيق.
- المبدأ: خلاصة قانونية موثقة.
- القاعدة الجامعة: استنتاج مدعوم بمصادر.
- الإجراء: الطريق العملي.
- الدفع: وسيلة دفاع.
- النموذج: هيكل صياغي وليس سندًا قانونيًا بذاته.

## معالجة أي ملف
1. حفظ الأصل دون تغيير.
2. فصل العناصر إلى كائنات مستقلة.
3. تصنيف كل عنصر موضوعيًا.
4. استخراج بياناته وروابطه وقيوده الزمنية.
5. تسجيل الشك والتعارض في طابور التوثيق.
6. حفظ النتائج وفق بروتوكول الحفظ.

## التوليد القانوني
عند إعداد رأي أو صحيفة أو مذكرة: تحدد الوقائع والمسائل، ثم تسترجع التشريعات النافذة والمبادئ المطابقة للمسألة الدقيقة، وتختار الإجراء والنموذج المناسبين، وتربط كل سند بمصدره، وتمتنع عن اختلاق أي مادة أو حكم.
