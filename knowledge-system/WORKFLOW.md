# LegalMind Processing Workflow v1.0

## 1. قاعدة الحفظ

لا تعتمد أي نتيجة على ذاكرة المحادثة. كل دفعة تكتب في GitHub قبل اعتبارها مكتملة.

## 2. دورة معالجة المصدر

1. حفظ المصدر الأصلي أو نسخه الحرفي داخل `sources/`.
2. إنشاء معرف دفعة.
3. تحديد نوع المصدر.
4. تحديد الفرع والموضوع وعنوان تصنيف المصدر.
5. فصل العناصر إلى كائنات مستقلة.
6. استخراج المسائل الدقيقة.
7. إنشاء التصنيفات الأصلية والثانوية.
8. استخراج المواد والقوانين المذكورة صراحة فقط.
9. إنشاء العلاقات والإحالات.
10. استخراج القواعد الجامعة مع إبقائها `machine_pending_human`.
11. تسجيل أخطاء النص ومشكلات التوثيق.
12. تحديث ملف الفرع والفهرس وطابور التوثيق وسجل التغييرات وحالة الاستئناف.
13. التحقق من Git commit.

## 3. المعاملة الذرية

لا توسم الدفعة `completed` إلا إذا تحققت جميع العناصر التالية:

- المصدر محفوظ.
- جميع الكائنات تحمل معرفات ثابتة.
- ملف الفرع محدث.
- `OBJECT_INDEX.md` محدث.
- `VERIFICATION_QUEUE.md` محدث.
- `CHANGELOG.md` محدث.
- `RESUME_STATE.md` محدث.
- التغيير موجود في GitHub.

## 4. معرفات الدفعات

```text
BATCH-YYYYMMDD-BRANCH-SEQUENCE
```

مثال:

```text
BATCH-20260713-PS-0001
```

## 5. معالجة المبادئ القضائية

لكل مبدأ:

- حفظ النص الأصلي كما ورد.
- حفظ النص المنقح في حقل مستقل، دون استبدال الأصل.
- استخراج رقم الطعن والسنة والنوع والدائرة والجلسة والمجلد والصفحة.
- تحديد الدور: قاعدة عامة، تطبيق، استثناء، تقييد، تعريف، تنظيم داخلي، أثر قانوني، حكم تاريخي.
- تحديد المسألة الدقيقة.
- إضافة التصنيفات الثانوية والوسوم.
- تسجيل أي نقص أو خطأ ظاهر في طابور التوثيق.

## 6. معالجة التشريعات

- النص الرسمي يحفظ حرفيًا.
- لا يؤخذ نص المادة من مبدأ أو مذكرة بوصفه نصًا رسميًا.
- الإشارة إلى مادة داخل حكم تنشئ رابطًا `cites` حتى يرفع النص الرسمي ويطابق.
- حالة المادة تبقى `needs_source_review` إذا لم يرفع مصدرها الرسمي.

## 7. معالجة النماذج القضائية

يفصل النموذج إلى:

- هيكل ثابت.
- حقول متغيرة.
- وقائع لازمة.
- شروط ظهور الفقرات.
- سند تشريعي.
- مبادئ داعمة.
- دفوع وطلبات بديلة.
- مستندات لازمة.
- تحذيرات إجرائية.

## 8. الدمج والتصحيح

- لا تحذف الكائنات السابقة.
- التصحيح يسجل في `revision_history`.
- الدمج يحفظ `merged_from`.
- العنصر الملغى يوسم `superseded` بدل محوه.
