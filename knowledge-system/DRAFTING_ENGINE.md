# LegalMind Drafting Engine v1.0

## 1. أنواع المخرجات

- رأي قانوني.
- صحيفة دعوى.
- مذكرة دفاع.
- صحيفة استئناف أو تمييز.
- طلب أو تظلم.
- إنذار.
- قائمة إجراءات ومستندات.

## 2. شروط بدء الصياغة

قبل الصياغة يحدد النظام:

- صفة كل طرف.
- الوقائع الثابتة وغير الثابتة.
- الطلب النهائي.
- المحكمة والمرحلة القضائية.
- الفرع والموضوع والمسألة الدقيقة.
- النصوص النافذة.
- المبادئ المطابقة.
- المستندات المتاحة والناقصة.

## 3. سلسلة السند

كل فقرة قانونية يجب أن ترتبط بأحد الآتي:

```text
تشريع رسمي
أو حكم/مبدأ موثق
أو قاعدة جامعة مرتبطة بمصادرها
```

لا يعد النموذج أو المذكرة السابقة مصدرًا للقاعدة، بل مرجعًا للهيكل والأسلوب.

## 4. بناء النموذج الديناميكي

```text
هيكل ثابت
+ حقول متغيرة
+ فقرات مشروطة
+ دفوع بديلة
+ طلبات أصلية واحتياطية
+ مستندات لازمة
```

لا تدرج فقرة مشروطة ما لم تتحقق وقائعها.

## 5. الاستشهاد

يذكر، بحسب المتاح الموثق:

- رقم الطعن.
- نوعه وسنته.
- الدائرة.
- تاريخ الجلسة.
- المجلد والصفحة أو مرجع النشر.
- نص المادة ورقم القانون عند وجود مصدر تشريعي رسمي.

## 6. منع الهلوسة

يحظر:

- اختلاق مادة أو حكم.
- نقل نص مادة من مبدأ بوصفه النص الرسمي.
- استخدام حكم تاريخي كقاعدة حالية دون قيد.
- تعميم تطبيق خاص.
- إخفاء التعارض أو نقص الوقائع.

## 7. المخرجات الإلزامية

كل صياغة نهائية تتضمن داخليًا أو في ملحق:

- المصادر المستخدمة.
- المصادر المستبعدة وسبب الاستبعاد عند أهميتها.
- الوقائع الناقصة.
- ملاحظات التوثيق.
- مستوى الثقة.
